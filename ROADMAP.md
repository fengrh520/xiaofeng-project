# 📅 小封机器人升级路线图 (Roadmap)

## 🚀 已完成功能 (Phase 1)
- [x] **基础架构**：Vue 3 + Django 5 + SQLite 前后端分离环境搭建
- [x] **AI 核心**：接入 Moonshot (Kimi) API 实现提示词优化
- [x] **用户认证**：JWT 注册/登录系统，数据安全性保障
- [x] **数据隔离**：用户只能访问自己的 Prompt 记录
- [x] **数据增强**：支持星标 (Star)、置顶 (Pin)、搜索、筛选
- [x] **UI 交互**：Element Plus 界面，历史记录管理

## 🛠 待开发计划 (Phase 2 - Next Steps)

### 1. 📊 数据可视化 (Data Visualization)
- **目标**：让用户直观看到自己的使用频率和习惯。
- **方案**：引入 [ECharts](https://echarts.apache.org/)。
- **功能点**：
  - 每日/每周/每月 提示词优化数量趋势图 (折线图/柱状图)。
  - 常用关键词词云 (Word Cloud)。
  - 提示词长度分布分析。

### 2. 🎨 交互体验升级 (UX Polish)
- **目标**：打造丝滑、现代化的使用体验。
- **功能点**：
  - **骨架屏 (Skeleton)**：加载数据时显示占位动画，拒绝白屏。
  - **Hover 动效**：按钮、卡片的微交互动画。
  - **快捷键支持**：`Ctrl+Enter` 立即提交优化。
  - **字数统计**：实时显示输入字符数。
  - **一键清空**：快速重置输入框。

### 3. 🧩 功能模块扩展 (Feature Expansion)
- **目标**：增加产品的实用性和丰富度。
- **功能点**：
  - **模板中心**：预设常用 Prompt 模板 (如“周报生成”、“代码解释”、“邮件润色”)，一键套用。
  - **优化评分**：让 AI 对原始 Prompt 进行打分 (0-100) 并给出改进建议。
  - **多场景选择**：在优化时选择特定场景 (学术、职场、创意)，让 AI 输出更精准。
  - **个人中心**：修改头像、昵称、密码管理。

### 4. ☁️ 部署与运维 (Deployment)
- **目标**：从本地开发走向线上服务。
- **计划**：
  - 数据库迁移至 MySQL/PostgreSQL。
  - Docker 容器化部署。
  - Nginx 反向代理配置。
